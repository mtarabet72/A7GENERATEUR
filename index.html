❓ Aide et documentation

📖 Guide d'utilisation

1. Onglet SAISIE : Saisissez les informations de base pour générer une affichette A4 (gencode, prix, unité, etc.).

2. Onglet FILE : Gérez la file d'attente des produits avant génération massive.

3. Onglet BASE : Sauvegardez vos produits fréquents dans une base de données.

4. Onglet HISTORIQUE : Consultez l'historique de toutes les affichettes générées.

5. Onglet TEMPLATES : Créez, modifiez ou supprimez des templates pour personnaliser la mise en page.

6. Onglet CONFIG : Définissez les valeurs par défaut et les paramètres de sécurité.

⌨️ Raccourcis clavier

  • Entrée : Ajouter à la file depuis le champ code produit
  • Ctrl + S : Sauvegarder les données
  • Ctrl + P : Générer le PDF

🔧 Dépannage

Si une affichette ne s'affiche pas correctement :

  • Vérifiez que tous les champs sont remplis.
  • Assurez-vous que les positions X/Y des éléments dans le template sont valides (entre 0 et 210 pour X, 0 et 297 pour Y).
  • Essayez de rafraîchir la page ou de vider le cache du navigateur.
