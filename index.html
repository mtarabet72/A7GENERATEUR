❓ Aide et documentation

📖 Guide d'utilisation

1. Onglet Base : Saisissez les informations de base pour générer une affichette A4 (gencode, désignations, prix, picto, etc.).

2. Onglet Files : Gérez les fichiers générés (PDF).

3. Onglet Templates : Créez, modifiez ou supprimez des templates pour personnaliser la mise en page des affichettes.

4. Onglet Configuration : Définissez les valeurs par défaut (secteur, picto, template).

5. Onglet Gestion de mot de passe : Protégez l'accès à l'application.

📌 Règles de calcul

  • Secteur Food : La différence (prix barré - prix promo) est affichée uniquement si elle est ≥ 5 DH.
  • Secteur NonFood : La différence est affichée uniquement si elle est ≥ 10% du prix barré.

🔧 Dépannage

Si une affichette ne s'affiche pas correctement :

  • Vérifiez que tous les champs sont remplis.
  • Assurez-vous que les positions X/Y des éléments dans le template sont valides (entre 0 et 210 pour X, 0 et 297 pour Y).
  • Essayez de rafraîchir la page ou de vider le cache du navigateur.
